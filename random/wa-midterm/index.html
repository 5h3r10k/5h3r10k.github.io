My Recommendations

Tinker further...

Roadmap.sh

Ever wanted to get into a specific technical field but didn't know where to start? Roadmaps is a comprehensive tool that provides a roadmap with suggestions on skills, projects, and learning resources!

W3 Schools

Learning a new programming language or framework can be daunting. W3Schools provides very detailed documentation and examples for all the current popular languages as well as other specialized programming languages.

edX

If you're more of a lecture-based learner, edX is a free platform that hosts popular technical courses from renowned universities such as MIT, Harvard, and Stanford! It allows you to learn invaluable university-level material conveniently.

GitHub

If you're thinking about starting your projects journey, you'll want to make sure your code is stored safely and easily accessible. GitHub is a cloud storage and code collaboration tool, with further automation tools for us geeks.

Stack Overflow

While programming and building, there are times when you might encounter a bug or be unsure of how to continue. If so, head over to Stackoverflow and just ask! A community of fellow hobbyists will happily help you out - and you can pay it forward by helping someone else out someday!

WordPress

After you've started creating cool things, you want to show them off! Wordpress allows you to create your own portfolio website which you can populate with various templates and content types.

Fritzing

When you're working with wiring and electronics, it's important to prototype and plan out your layout and wiring. Fritzing is a tool that allows you to create and simulate circuit drawings, as well as export files to print your own circuit board!
