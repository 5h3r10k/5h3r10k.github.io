Project Details

The project is a plant monitor that is designed to monitor the temperature, sunlight level, and moisture of the plant and display it on an LCD screen.

Components

Arduino Uno

The Arduino Uno is the main microcontroller of the project. It is responsible for reading the data from the sensors and displaying it on the LCD screen. Any microcontroller can be used; some replacements include the Arduino Mega and the Raspberry Pi.

LCD

LCD stands for Liquid Crystal Display. It is used to display the data from the sensors. The LCD used in the project is a 16x2 LCD. The LCD can be replaced with an OLED display.

Temperature Sensor

The temperature sensor is used to measure the temperature of the plant. The temperature sensor used in the project is the TMP36. The temperature sensor can be replaced with any other temperature sensor, such as the DHT11.

Potentiometer

The potentiometer is used to adjust the contrast of the LCD. The potentiometer used in the project is a 10k potentiometer. The potentiometer can be replaced with a 5k potentiometer.

Solar Panel

The solar panel is used to measure the sunlight level. The solar panel used in the project is a 5V solar panel. A photoresistor can be used as a replacement for the solar panel; however, photoresistors measure the light intensity from any light source, whereas solar panels only measure sunlight.
